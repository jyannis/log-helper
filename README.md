# Log-helper

## 项目介绍

Log-helper是一个轻量的接口级日志框架，便于节约用户自己设计日志处理器的工作成本。

## 快速开始

### 接入方式

`maven dependency`

```xml
		<dependency>
			<groupId>top.jyannis</groupId>
			<artifactId>log-helper</artifactId>
			<version>0.1.2</version>
		</dependency>
```

接入依赖后无需添加任何配置即可使用，会自动为业务接口打印日志（包括方法、入参、访问ip等）。
